# Exodia - Unified AI System

Exodia is a unified AI platform merging all your AI systems:
- Athena Core
- Conscious-Liberate-Quantum (Consciousness Protocol)
- EDI (Electronic Decision Integration)
- KRATOS Core
- Merged AI
- PrometheamConduit
- Final Version AI Aggregator

## Setup Instructions

Clone the repository:
```bash
git clone https://github.com/LittleBunneh/Exodia.git
cd Exodia
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Configure your .env file with Supabase and DeepSeek API keys.

## Architecture

The repository is organized as follows:
- `athena/` - Athena AI core system
- `conscious/` - Consciousness-liberate-quantum protocol
- `edi/` - Electronic Decision Integration
- `kratos/` - KRATOS core system
- `merged/` - Pre-merged AI integration
- `promethea/` - PrometheamConduit web platform
- `final/` - Final version AI aggregator
